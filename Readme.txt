--- SISTEMA DE CADASTRO DE CLIENTES ---

Este sistema desenvolvido com PHP e MySQL é um sistema simples de gerenciamento de cadastro de clientes, com inserção, edição, visualização e exclusão de registros de clientes.


--- COMO INSTALAR O SISTEMA ---

1. Clone o repositório para o diretório do seu servidor web (ex: 'htdocs' para XAMPP).

2. Importe o arquivo 'database.sql' para criar a tabela necessária no MySQL.

3. Configure as credenciais do banco de dados em 'config/database.php'.


--- COMO UTILIZAR O SISTEMA ---

1. Acesse a aplicação através do navegador: http://localhost/seu_diretorio/index.php

2. Para cadastrar um novo cliente: Clique em "Cadastrar Novo Cliente" e preencha o formulário, carregue uma foto do cliente e clique em "Enviar" para cadastrar no banco de dados.

3. Para editar os dados dos clientes cadastrados: Clique em "Editar" ao lado do cliente na lista para modificar os dados cadastrado.

4. Para excluir os dados dos clientes cadastrados: Clique em "Excluir" ao lado do cliente na lista para remover o registro cadastrado.
